預估等候時間僅供參考，請以現場叫號為準。

費用

NT$500 / 人（或寵物一隻）

NT$1,000 / 兩人，或人＋寵物一隻

產出形式：小木板現場繪製，附親筆簽名，完成即可帶走。

參加方式

  1. 開店起即可購票領號，費用於結帳時收取。
  2. 結帳後由店員發放實體號碼牌（附 QR Code）。
  3. 掃描 QR Code 查看當前叫號進度，自行估算等候時間。
  4. 畫寵物者請事先備妥清晰照片（正面或側面均可）。
  5. 前方剩餘 3–5 位時，請返回活動區報到。
  6. 號碼牌發完即額滿，不接受加號。

過號處理

號碼叫到時若不在場，視為過號。

過號者返場後，將安排插入第四順位候補。

【注意事項】

  • 預估等候時間僅供參考，實際進度以現場叫號為準。
  • 號碼牌遺失者，請持購買收據向店員核對；無法核實者不予補號。
  • 寵物請以手機照片代替本體入店，避免違反百貨規定。
  • 退款：老師尚未開始繪製前因故放棄者，退款與否由現場主管決定。
  • 活動因不可抗力因素暫停或終止時，退款事宜由現場主管公告處理，主辦方保留最終決定權。
  • 頁面叫號進度由門市人員手動更新，數據僅供參考，不構成任何等候時間之保證。
